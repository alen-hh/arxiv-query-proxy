üìñ API Usage

Endpoint: /api/arxiv-query

Method: GET

Parameters:

search_query
Required. Your search terms (e.g., "machine learning", "quantum computing", "neural networks")
max_results
Optional. Number of results to return (default: 10, max: 100)
start
Optional. Starting index for pagination (default: 0)
sortBy
Optional. Sort by "submittedDate" or "relevance" (default: "submittedDate")
sortOrder
Optional. "ascending" or "descending" (default: "descending")

Example URLs:

  • /api/arxiv-query?search_query=machine learning
  • /api/arxiv-query?search_query=quantum computing&max_results=20
  • /api/arxiv-query?search_query=neural networks&sortBy=relevance
